2. 外部コラボレーターの扱い

  • Freeプラン: プライベートリポジトリに無制限に外部コラボレーターを招待でき、料金はかからない。
  • Teamプラン: 外部コラボレーターをプライベートリポジトリに招待する場合、そのユーザー分のライセンス料金がかかる。パブリックリポジトリへの招待は無料。

3. GitHub Packages 📦

GitHub Packagesは、ソフトウェアパッケージやコンテナイメージなどを、ソースコードと同じGitHubリポジトリ内で直接ホスト・管理できるサービス。npmレジストリやDocker Hubのようなパッケージ管理の仕組みを、GitHubに統合したものと考えると分かりやすい。

主な利点

  • 一元管理: ソースコード、Issue、CI/CDワークフロー (Actions)、そしてパッケージ（成果物）をすべてGitHub上で管理できる。
  • シームレスな連携: GitHub Actionsとの連携が非常にスムーズで、ビルドから公開までのCI/CDパイプラインを簡単に構築できる。
  • 統一された権限管理: パッケージのアクセス権はリポジトリの権限と連動するため、認証情報の管理が簡素化される。

サポートされている主なパッケージ形式

Docker, npm, Maven, Gradle, NuGet, RubyGemsなど、さまざまな言語や環境の主要なパッケージ形式に対応している。

料金について

料金は主にストレージ（容量）とデータ転送量（ダウンロード量）に基づく。パブリックリポジトリのパッケージは完全に無料で、プライベートリポジトリ向けには各プラン（Free, Team, Enterprise）に一定の無料枠が用意されている。

4. コードオーナー & 必須レビュー

レビュープロセスを自動化し、コードの品質を担保する機能です。主にTeamプラン以上で利用できる。

コードオーナー (Code Owners)

リポジトリ内のコードの特定部分に対して、責任を持つ担当者を定義する機能。.github/CODEOWNERSファイルで設定する。

# docsディレクトリ以下のファイルは @docs-team が担当
/docs/    @docs-team

# .jsで終わるファイルは @javascript-devs が担当
*.js      @javascript-devs

必須レビュー (Required Reviews)

プルリクエストがブランチにマージされる前に、指定した条件のレビューが完了していることを強制する「ブランチ保護ルール」の一種。「コードオーナーからのレビュー」を必須にすることで、より強力なレビュー体制を築ける。

5. 保護されたブランチ

重要なブランチを意図しない変更から守る仕組みです。プランが上位になるほど、より厳格なルールを設定できる。

6. セキュリティと認証機能

SAML SSO (シングルサインオン)

一度のログインで、複数の異なるサービスを利用できるようにする仕組み。主にEnterpriseプランで提供され、組織のID管理システム（IdP）と連携する。Google Workspace (旧 G Suite) をIdPとして利用することも可能。

高度な監査ログ

「いつ、誰が、何をしたか」をより詳細に記録・追跡する機能。ログの長期保持や、外部監視システムへのリアルタイム転送（ストリーミング）が可能。

GitHub Advanced Security (GHAS)

開発の早期段階で脆弱性を発見・修正するための統合セキュリティツール群です。Teamプランではアドオンとして購入可能（2025年4月より製品分割）。

1. GitHub Secret Protection: $19 /月（アクティブなコミッター1人あたり）

  • シークレットスキャン (Secret scanning): コード内のAPIキーやパスワードを検出
  • プッシュ保護 (Push protection): 機密情報がリモートリポジトリにプッシュされるのをブロック

2. GitHub Code Security: $30 /月（アクティブなコミッター1人あたり）

  • コードスキャン (Code scanning): 静的解析エンジン「CodeQL」によるコードの脆弱性スキャン。
  • 依存関係レビュー (Dependency review): オープンソースライブラリの脆弱性チェック。
  • Copilot Autofix: AIによる脆弱性の自動修正提案。

合計料金の考え方

Teamプランの基本料金に、下記のアドオン料金が追加されます。

合計月額料金 = (Teamプランのユーザー料金) + (アドオンの料金 × アクティブなコミッター数)

例えば、Teamプラン（$4/ユーザー/月）を利用しているチームで、5人がアクティブなコミッターとして該当し、「Secret Protection」だけを追加する場合、アドオンの料金は $19 × 5人 = $95 となる。

必要な機能に応じて、片方だけ、あるいは両方のアドオンを購入するかを選択できる。
